# Java Version of Employee SRP Practice

## Installion

```shell script
$ mvn clean install
```

* Run tests:
```shell script
$ mvn test
```

## File Structure

* `Employee.java`: Main Class
* `EmployeeTest.java`: Test files


## Environment

* jdk: openjdk 13.0.2 2020-01-14

You can set `Project Structure -> Project` and `Project Structure -> Modules` to 13.
